# Changelog

All notable changes to Clarity for Facebook will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-26

### Added
- Initial release of Clarity for Facebook
- Clean Mode master toggle for all features
- Remove Reels from feed
- Remove sponsored posts and ads
- Remove "Suggested for you" posts
- Remove Marketplace advertisements
- Remove search ads
- Remove "Follow" suggestions
- Remove "People you may know" suggestions
- Settings export/import functionality
- Cross-browser support (Chrome, Edge, Firefox)
- Settings sync across devices
- Multilingual support for content detection
- Performance-optimized DOM observation
- React-based popup UI with Tailwind CSS
- TypeScript strict mode implementation
- Comprehensive error handling
- Smooth animations and transitions

### Technical Details
- Manifest V3 implementation
- MutationObserver with throttling
- Debounced batch processing
- WebExtension Polyfill for cross-browser compatibility
- Webpack 5 build system
- ESLint and TypeScript configuration
- Modular component architecture
